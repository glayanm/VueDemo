判斷顯示

判斷是否顯示,show 只會修改 disabled 狀態,if else 則是改變Dom ,若要經常性變化請使用 show .
判斷狀態石可在v-show="(JavaScript)" 中寫入簡單的JavaScript, ex. v-show="1===1"
或是有布林值的 data 等等.

  • Demo
  • Html Code
  • Java Script Code
v-if Show Time
v-else Show Time
v-show Show Time

    <div class="col-sm-12 col-md-12 col-lg-12 form-group row">
        <div class="m-3">
            <input id="vIfDemo" class="form-control btn btn-primary" value="SwitchIf" type="button"
                   v-on:click="vIfDemo = !vIfDemo" />
        </div>
    </div>
    <h5 v-if="vIfDemo">v-if Show Time</h5>
    <h5 v-else="vIfDemo">v-else Show Time</h5>
    <h5 v-show="vIfDemo">v-show Show Time</h5>


    <script>
       const app = {
            data() {
                return {
                    vIfDemo: true,
                }
            },
            methods: {
                showChapter: function (index) {
                    return this.chapter === index;
                }
            }
        }
        
        Vue.createApp(app).mount("#app");
    </script>
